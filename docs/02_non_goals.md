# Non-goals

Archive Recall is NOT:
- a DRM circumvention tool
- a piracy tool
- a hacking framework
- a stealth or covert tool
- a "download anything from anywhere" product

Archive Recall will not:
- bypass platform protections
- break encryption on protected media
- claim compliance while doing non-compliant actions

Archive Recall may store:
- user-owned files
- user-created files
- lawfully obtained DRM-free media
- metadata and references to platform content (IDs, links, playlists, titles) via official exports/APIs
