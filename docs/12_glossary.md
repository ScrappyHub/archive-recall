# Glossary

- Archive: preserved canonical objects (hash-identified)
- Recall: deterministic restoration/load of archived state
- Object: a single content-addressed unit (sha256 identity)
- Tree: folder representation (Merkle)
- Event: append-only action record
- Transformer: approved, versioned operation with schema + policy constraints
- Load Plan: staged set of writes + verifications + commit
- Checkpoint: rollback-able state capture for targets
